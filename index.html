Objetos en JS

En este archivo se muestran algunos ejemplos de objetos. Para ver su funcionalidad solo tienes que entrar a la consola del navegador.

Sí quieres agregar capítulos a un libro, puedes hacerlo con el siguiente comando:

    libro1.agregarCapitulo("Nombre del Nuevo capitulo");

Sí quieres eliminar un capítulo de un libro, puedes hacerlo con el siguiente comando:

    libro1.eliminarCapitulo("Nombre del capitulo a eliminar");

Sí quieres mostrar los capítulos de un libro, puedes hacerlo con el siguiente comando:

    libro1.mostrarCapitulos();

Posdata: El número que acompaña a la primera palabra (libro) en el código es el número de libro
